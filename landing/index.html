Current App Scope

One compact window, four focused views.

The current 0.3.4 app keeps the product narrow: prompt polishing before you send, instant explanation after you read, settings that match your workflow, and local history for reuse.

Prompt Polish

Turn Chinese or mixed-language drafts into direct, natural prompts for Codex, Claude Code, Cursor, or any AI coding assistant.

  • Conservative and enhanced rewrite modes
  • Diff highlighting for every change
  • Keyboard-first copy and quick submit

Instant Explain

Select unfamiliar English from AI output and open a lightweight explanation with meaning, context, and alternate phrasing.

  • Meaning in your native language
  • Technical context instead of generic translation
  • Hotkey trigger with clipboard fallback

History and Reuse

Copied and confirmed results are saved locally so useful phrasing, explanations, and favorites stay close at hand.

  • Recent prompt polish and explain items
  • Favorites for expressions worth keeping
  • Local storage with no cloud sync
How It Works

A shorter loop around AI coding.

01

Summon LingoKey

Press the main hotkey. A compact always-on-top window opens with the input already focused.

Ctrl + Shift + L
02

Polish the Prompt

Write in Chinese, English, or a mix. LingoKey rewrites it into your configured learning language while preserving technical intent.

03

Copy, Send, and Save

Copy the result into your AI coding assistant. Confirmed results are added to local history for quick reuse.

04

Explain on Demand

When AI output contains an unfamiliar phrase, select it and press the explain hotkey for native-language context.

Ctrl + Shift + E

Built for keyboard-first developers

Main, explain, and settings shortcuts are customizable. The app also supports theme, opacity, always-on-top, auto-start, language pair, and provider settings.

Polish View
Ctrl + Shift + L
Explain View
Ctrl + Shift + E
Settings
Ctrl + Shift + S
Get Started

Use your own AI provider.

LingoKey is open source and free. It works with OpenAI-compatible APIs, defaults to DeepSeek, and keeps your API key in local settings.
